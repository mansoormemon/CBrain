Detailed Description

A simple 2D image.

Limitations:

  1. All channels will have the same depth i.e. u8.
  2. Color model will be inferred based on the number of channels as follows:
    Channels Associated Color Model
    1 Gray
    2 Gray Alpha
    3 RGB
    4 RGBA
    Note: Anything besides these will render the image incompatible with most of the library functions.
