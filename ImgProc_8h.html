◆ CBImgProcIsCompatible()

bool CBImgProcIsCompatible ( CBImage *  img,
CBIPColorModel  model 
)

Checks if the given image is compatible with the said color model.

Parameters
imgthe image to be checked.
modelthe model with which the compatibility is to be checked.
Returns
  • true if the image is compatible with the given model, false otherwise.

model can also be a bit-field.

For example:

// Checks if the image is compatible with any of the two models.
bool isCompatible = CBImgProcIsCompatible(img, CBIPCM_GRAY | CBIPCM_RGB);
