Detailed Description

Contains all the basic functionality for working with images.

Available functionality:

  1. Functions for creating and deleting images.
  2. IO functions for reading and writing image.

  3. Utilities:

    • Functions for cloning images.
    • Functions for extracting sub-image from an image.
    • Accessors and Mutators.
    • Query functions.
    Since
    v0.0.1
