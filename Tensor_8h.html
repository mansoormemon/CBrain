Detailed Description

Contains functionality to deal with n-dimensional generic vectors.

Available functionality:

  1. Create and delete tensor.
  2. Reshape tensor.
  3. Utilities:
    • Accessors and mutators.
    • Query functions.
Since
v0.0.1.

◆ CBTensorReshape()

CBTensor* CBTensorReshape ( CBTensor *  tensor,
i32  dims,
  ... 
)

Reshape a given tensor based on the given parameters.

Parameters
tensorthe tensor to be reshaped.
dimsthe new dimensions of the tensor.
...new shape of tensor i.e. the size of each dimension individually.
Returns
  • tensor on success, nil on failure.
Note
No buffer (re)allocation will take place during reshaping. Reshaping will only be performed if the new dimensions are compatible with the old ones i.e. total number of elements remain the same.

Usage:

// Create a 3D tensor of shape 3D:28x28x3 of type u8.
// Total elements = 2352.
CBTensor *tensor = CBTensorFrom(u8, 3, 28, 28, 3);
// Reshaping tensor, so the new shape becomes 6D:2x2x3x4x7x7.
// Total elements = 2352.
CBTensorReshape(tensor, 6, 2, 2, 3, 4, 7, 7);
